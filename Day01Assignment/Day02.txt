Difference between documents and windowobjects:

Document object:
1. It represents the document loaded inside the window or browser
2. The properties related to it are stored in the document object
3. It is oaded after the loading window because tehe window contains a document
4. It is the root elememt  of the document model
5. It is an object of the window
6. Properties:
            element.innerHTML = new html content
7. Methods:
            document.createElement(element)

Window Object:
1. It represents the browser window in which you are seeing the content
2. The properties related to it are stored in the window object
3. It is loaded before the document because window container document
4. The window is the global element for all the objects, functions, etc.
5. It is an object of the browser
6. Properties:
                window.innerheight
7.Methods:
            window.open()
